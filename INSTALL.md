# Установка и запуск SonarCore

## Требования

- Python 3.8 или выше
- pip

## Установка зависимостей

### Создание виртуального окружения (рекомендуется):

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

### Или установка пакета в режиме разработки:

```bash
pip install -e .
```

## Запуск приложения

### С использованием виртуального окружения:

```bash
# Активируйте виртуальное окружение (если ещё не активировано)
source .venv/bin/activate

# Запустите приложение
python main.py
```

### Или используйте скрипт запуска:

```bash
./run.sh
```

### Или напрямую через GUI модуль:

```bash
source .venv/bin/activate
python -m gui.main_window
```

## Структура проекта

```
SonarCore/
├── core/           # Вычислительное ядро
├── data/           # База данных параметров
├── gui/            # Пользовательский интерфейс
├── tests/          # Тесты
├── logs/           # Логи (создаётся автоматически)
└── main.py         # Точка входа
```

## Использование

1. Запустите приложение
2. Выберите параметры аппаратуры (трансдусер, LNA, VGA, ADC)
3. Задайте параметры CHIRP-сигнала
4. Установите параметры среды (температура, солёность, глубина)
5. Задайте диапазон дальности
6. Нажмите "Запустить симуляцию"
7. Просмотрите результаты и рекомендации
8. При необходимости используйте "Оптимизировать" для автоматической оптимизации параметров

## Добавление данных

Добавьте файлы JSON в соответствующие директории:
- `data/transducers/` - параметры трансдусеров
- `data/lna/` - параметры LNA
- `data/vga/` - параметры VGA
- `data/adc/` - параметры ADC

Формат файлов см. в примерах в соответствующих директориях.
